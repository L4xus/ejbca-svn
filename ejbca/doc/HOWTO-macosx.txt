
Running EJBCA on MacOSX
========================
This example uses MySQL as database.

$ cd /usr/local/ejbca
$ sudo ant replaceDS

- Say mysql for Type of database
- Say java:/EjbcaDS for data source

$ sudo ant deploy

- Restart JBoss
- After installation you will probably need to modify the JBoss Tomcat configuration file in server/default/deploy/jbossweb-tomcatxx.sar/server.xml
since some paths and parameters may differ. So save the original server.xml file to compare later on.

$ sudo chmod u=rwx install.sh
$ sudo chmod u=rwx ca.sh
$ sudo chmod u=rwx setup-adminweb.sh
$ sudo chmod u=rwx setup.sh
$ sudo chmod u=rwx batch.sh

- Comment out the 2 "keytool"-rows in setup-adminweb.sh
- Comment out the "rm tmp/rootca.der"-row in setup-adminweb.sh

$ sudo ./install.sh <javacacerts password>
