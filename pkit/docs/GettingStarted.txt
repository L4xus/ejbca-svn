Assuming you have your java environment and ant installed

if you downloaded pkit-noprov.tgz ...
You still need the bouncy castle provider jar copy yours into pkit's lib directory or download one to there from
www.bouncycastle.org

If you are building from source ...
run ant target "all"
run ant targets "makeCACert"   -to make a cert you can use to sign (or you can use the dummy one provided (keystore.p12))
run the example.sh script (or make a similar .bat file and run that and if you do then mail it to me

If you have a binary distribution ...
run the example.sh script (or make a similar .bat file and run that and if you do then mail it to me)

successfull execution of example.sh means you have your environment set up OK
